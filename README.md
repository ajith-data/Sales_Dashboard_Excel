# Excel Data Visualization Dashboard

## Project Overview

This project is an interactive Excel dashboard for analyzing and visualizing business data. It transforms raw data into clear insights using KPIs, charts, and slicers.

## Objectives

* Convert raw data into insights
* Build a one-screen interactive dashboard
* Track KPIs
* Enable dynamic analysis with slicers
* Present insights in a clear format

## Tools & Technologies

* Microsoft Excel (2016 / 2019 / 365)
* Pivot Tables & Charts
* Slicers & Filters
* Excel Functions (SUMIFS, COUNTIFS, IF, VLOOKUP/XLOOKUP)

## Dashboard Features

* KPI cards for key metrics (Sales, Profit, Orders, etc.)
* Interactive slicers
* Monthly and yearly trend analysis
* Category-wise and region-wise insights

## How to Use

1. Download the Excel file
2. Open in Microsoft Excel
3. Enable editing
4. Use slicers to explore data
5. Analyze KPIs and trends

## Key Learnings

* Building professional Excel dashboards
* Designing dashboards to fit one screen
* Creating meaningful KPIs
* Using Pivot Tables and slicers for interactive analysis
* Improving data storytelling with visuals
